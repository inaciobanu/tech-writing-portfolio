## Before You Start

You'll need:
- A PayFlow account ([sign up free](https://payflow.io/signup))
- Your **test API key** from the Dashboard
- Basic familiarity with REST APIs and your language of choice

:::info Test mode is safe
All requests made with a `sk_test_` key are completely sandboxed. No real payments are processed.
:::
